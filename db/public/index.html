Endpoint de Validação

Use esta rota no seu app/script para validar keys dos seus usuários.

POST /api/auth

BODY (JSON):

{ "key": "Key-XXXXX-XXXXX-XXXXX-XXXXX", "hwid": "identificador-unico-do-pc", "mb": "fabricante-placa-mae" // opcional }

RESPOSTA (sucesso):

{ "valid": true, "tipo": "mensal", "expiracao": "2025-07-06T..." } { "valid": false, "reason": "Key expirada" }

EXEMPLO (Python):

import requests, uuid, platform res = requests.post("/api/auth", json={ "key": input("Cole sua key: "), "hwid": str(uuid.getnode()), "mb": platform.node() }) data = res.json() if data["valid"]: print("Acesso liberado!", data["tipo"]) else: print("Acesso negado:", data["reason"])
